# BuffNotifier
バフの残り時間が切れた時に音を鳴らします
あまり画面は洗練されていないです
# 使い方
白チャットで`/bn`と入力します。    
設定画面が開きます。  
Addを押すと項目が追加されます。  
左端の欄はバフIDです。https://tos-jp.neet.tv さん辺りから探してください。 
中間の項目はバフが切れた時に再生する音です。一覧のほとんどの効果音はなりませんが何とか探してください。
設定が終わったらApplyを押すと適用されます 。 
削除したい場合はDeleteを押してください。  
なお、設定は全キャラで共通です。
